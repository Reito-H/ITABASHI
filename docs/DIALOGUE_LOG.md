### 設計決定事項

| 決定項目 | 決定内容 | 理由 |
|---------|---------|------|
| インフラ | Cloudflare Workers + D1 | 既存CF環境と統合、低コスト |
| DB | SQLite (D1) | Workers相性最良 |
| フロントエンド | Vanilla JS + Alpine.js + Tailwind | シンプル、Workerと相性良 |
| 招待コード | 6桁英数字、有効期限7日 | セキュリティと利便性のバランス |
| LINE紐付け | 招待コード方式 | 第三者なりすまし防止 |
| 月度定義 | 18日スタート17日締め | ユーザー指示通り |
