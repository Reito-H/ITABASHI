# Benten管理システム

タクシー会社（弁天クラブ）の社員管理Webシステム。新人の離職防止（旧称: 新人離職防止システム）から始まり、現在は社員管理・シフト・報告・LINE連携まで扱う統合管理システムに拡張されている。

## 主な機能

- **社員管理** — 登録・編集・ステータス管理・CSVインポート・絞り込み検索
- **新人シフト管理** — 研修スケジュール（午前/午後/研修担当）・個人予定表の印刷
- **班長シフト** — 班長・指導者の月間シフト（LIFFからの希望提出・0時LINE通知）
- **総合新人管理** — 新卒Info・フォローリスト・面談管理
- **売上管理** — 日別営業収入・乗車回数・ODO記録（LIFF入力対応）
- **報告センター** — 忘れ物・事故・違反の報告（LIFF）と管理画面での進捗管理
- **車両検索** — 無線番号/ナンバー末尾での照会（Web・LINE両対応）
- **点検管理** — 車両点検スケジュール（写真AI取込あり・要目視照合）
- **マニュアルBot** — 登録マニュアルに基づくAIチャット
- **ベンテンクラブ シフト** — 会員シフト管理・LINEグループ自動送信
- **お知らせ配信 / LINE利用状況 / アカウント別権限**

## システム構成

| コンポーネント | 技術 |
|--------------|------|
| ホスティング / API | Cloudflare Workers（Hono・サーバーサイドHTML生成） |
| DB | Cloudflare D1（ローカルは `wrangler dev`） |
| LINE連携 | LINE Messaging API + LIFF |
| ドメイン | bentenclub.com |

開発コマンドは `system/package.json` を参照（`npm run dev` / `npm run deploy` など）。

## 月度ルール

**17日締め・18日スタート**（月ごとに 設定 → 月度設定 で変更可能）
例: 「6月度」= 5月18日〜6月17日
