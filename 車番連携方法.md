# LINE アカウント連携方法

## 役割の種類

| 役割 | できること |
|---|---|
| 統括管理者 | 全機能（車番検索・忘れ物・事故報告） + Web管理画面で権限変更 |
| 運行管理者 | 車番検索・忘れ物対応フォーム・事故報告フォーム |
| 車番管理者 | 車番検索のみ |
| 新人 | 売上記録・嫌なこと報告・シフト確認 |

---

## 連携手順

### 統括管理者

1. Benten管理LINEに `統括管理者登録` と送信
2. 氏名（漢字フルネーム）を送信
3. 統括管理者パスワードを送信
4. 「統括管理者で登録されました」と返信が来たら完了

---

### 運行管理者

1. Benten管理LINEに `運行管理者登録` と送信
2. 氏名（漢字フルネーム）を送信
3. 運行管理者パスワードを送信
4. 「運行管理者で登録されました」と返信が来たら完了

**登録後にできること：**
- 数字を送信 → 車番（無線番号・ナンバー）を検索
- `忘れ物対応` と送信 → フォームのリンクが届く（10分有効）
- `事故報告` と送信 → フォームのリンクが届く（10分有効）

---

### 車番管理者

1. Benten管理LINEに `車番連携` と送信
2. 氏名（漢字フルネーム）を送信
3. 車番管理者パスワードを送信
4. 「車番管理者で登録されました」と返信が来たら完了

**登録後：** 数字を送信するだけで車番検索できます。  
例）`6677` と送ると → 無線番号6677の車両情報が返ってくる

---

### 新人

管理者から発行された **招待コード（6桁英数字）** をLINEに送信するだけで完了です。  
コマンドは不要です。

---

## 連携解除

LINEで `れんけいかいじょ` と送信 → 登録情報が全削除されます。  
再連携する場合は上記の手順を最初からやり直してください。

---

## トラブル対応

| 症状 | 対処 |
|---|---|
| 「パスワードが正しくありません」と返ってきた | コマンドから入力し直す（例：`車番連携` → 名前 → パスワード） |
| 操作の途中でわからなくなった | `キャンセル` と送信するとリセットされます |
| フォームのURLが「期限切れ」と表示される | LINEに戻って `忘れ物対応` や `事故報告` を再送信する（リンクは10分で失効） |
| 登録したが機能が使えない | 統括管理者にWeb管理画面で権限を確認してもらう |
| 自分のLINE UIDを確認したい | `uid` と送信 |

## 開発者向け：パスワードの設定・変更

各パスワードは Cloudflare Workers の Secret として管理しています。

```bash
npx wrangler secret put LINE_REG_PWD_VEHICLE     # 車番管理者パスワード
npx wrangler secret put LINE_REG_PWD_OPERATIONS  # 運行管理者パスワード
npx wrangler secret put LINE_REG_PWD_GENERAL     # 統括管理者パスワード
```

パスワードを変更した場合、コードの変更・デプロイは不要です（Secretを更新するだけ）。
